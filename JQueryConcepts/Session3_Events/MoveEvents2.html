Mouse Events in JQuery 

    In jQuery, mouse events are used to handle user interactions with the mouse (clicking, hovering, moving, pressing, etc.).
    Here are some commonly used mouse events in jQuery:
1. click()
    Triggered when an element is clicked.
      Example: $("#myButton").click(function() { alert("Button clicked!"); });    
2. dblclick()
    Triggered when an element is double-clicked.
      Example: $("#myDiv").dblclick(function() { $(this).css("background-color", "yellow"); });   
3. mouseenter()
    Triggered when the mouse pointer enters the area of an element.
      Example: $("#myDiv").mouseenter(function() { $(this).css("background-color", "lightblue"); });  
4. mouseleave()
    Triggered when the mouse pointer leaves the area of an element.
      Example: $("#myDiv").mouseleave(function() { $(this).css("background-color", "white"); });  
5. mousemove()
    Triggered when the mouse pointer moves within the area of an element.
      Example: $("#myDiv").mousemove(function(event) { $("#coords").text("X: " + event.pageX + ", Y: " + event.pageY); });  
6. mousedown()
    Triggered when a mouse button is pressed down on an element.
      Example: $("#myDiv").mousedown(function() { $(this).css("border", "2px solid red"); });
7. mouseup()
    Triggered when a mouse button is released over an element.  
      Example: $("#myDiv").mouseup(function() { $(this).css("border", "none"); });
8. hover(handlerIn, handlerOut)
    A shorthand for mouseenter and mouseleave. It takes two functions: one for when the mouse enters, and one for when it leaves.
      Example: $("#myDiv").hover( function() { $(this).css("background-color", "lightgreen"); }, function() { $(this).css("background-color", "white"); } );
9. contextmenu()
    Triggered when the right mouse button is clicked (to open the context menu).
      Example: $("#myDiv").contextmenu(function(event) { event.preventDefault(); alert("Right-click detected!"); });

Summary:
  
click() → single click
dblclick() → double click     
mouseenter() → mouse enters element
mouseleave() → mouse leaves element
mousemove() → mouse moves within element  
mousedown() → mouse button pressed down
mouseup() → mouse button released
hover() → mouse enters and leaves element
contextmenu() → right-click (context menu)

Explanation: 

 
1. We create a div with the class "box" and id "myBox" to demonstrate mouse events.
2. We use jQuery to attach various mouse event handlers to the div: 
    - click(): Shows an alert when the box is clicked.
    - dblclick(): Toggles the "highlight" class on double-click.
    - mouseenter(): Changes background color when the mouse enters the box.
    - mouseleave(): Resets background color when the mouse leaves the box.
    - mousemove(): Updates and displays the current mouse coordinates within the box.
    - mousedown(): Changes border color when a mouse button is pressed down.
    - mouseup(): Resets border color when the mouse button is released.
    - hover(): Changes background color on hover and resets it when the mouse leaves.
    - contextmenu(): Prevents the default context menu and shows an alert on right-click. 
3. Each event handler uses jQuery methods like css() and toggleClass() to manipulate the box's appearance based on user interactions.
