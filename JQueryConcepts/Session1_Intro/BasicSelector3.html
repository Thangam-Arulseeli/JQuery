Basic Selectors - Introduction 

  Introduction

    In jQuery, selectors are used to "find" (select) HTML elements so we can work with them.
    The syntax is very simple:

        $("selector")
            Here are the three most common basic selectors:
                $("p") → selects all elements.
                $("#id") → selects element by id.
                $(".class") → selects elements by class name.

Explanation:

           $("#btn3") ->  Selects the element with id = "btn3" (usually a button).
            .click(function(){ ... }) ->  Attaches a click event handler to that button.
                    So, whenever the button is clicked, the function inside will run.
            $(".important") ->  Selects all elements that have the class "important".
            .addClass("highlight") -> Adds the CSS class "highlight" to all those selected elements.
            This usually changes the style/appearance of those elements (based on how "highlight" is defined in CSS).
