Filtering Selectors in jQuery 

    Filtering selectors are used to select a subset of elements from a larger set of matched elements.
    They help refine your selection based on specific criteria.

    Here are some commonly used filtering selectors in jQuery:  
1. :first
    Selects the first element in a set of matched elements.
      Example: $("p:first") selects the first 
 element.
2. :last
    Selects the last element in a set of matched elements.
      Example: $("p:last") selects the last 
 element.
3. :even
    Selects all even-indexed elements in a set of matched elements (0-based index). 
      Example: $("li:even") selects all even-indexed 
  •  elements (0, 2, 4, ...).
4. :odd
    Selects all odd-indexed elements in a set of matched elements (0-based index).
      Example: $("li:odd") selects all odd-indexed 
  •  elements (1, 3, 5, ...).
5. :eq(index)
    Selects the element at the specified index in a set of matched elements (0-based index
      Example: $("li:eq(2)") selects the third 
  •  element.
6. :gt(index)
    Selects all elements with an index greater than the specified index (0-based index).
      Example: $("li:gt(2)") selects all 
  •  elements with an index greater than 2 (3, 4, 5, ...).
7. :lt(index)
    Selects all elements with an index less than the specified index (0-based index).
      Example: $("li:lt(2)") selects all 
  •  elements with an index less than 2 (0, 1).
8. :not(selector)
    Selects all elements that do not match the specified selector.
      Example: $("p:not(.intro)") selects all 
     elements that do not have the class "intro".
