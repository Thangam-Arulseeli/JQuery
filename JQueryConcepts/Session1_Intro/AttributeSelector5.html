Attribute Selectors in jQuery 

    Attribute selectors are used to select elements based on the presence or value of a specific attribute.
    They are very useful when you want to target elements with certain attributes or attribute values.

    Here are some commonly used attribute selectors in jQuery:    
1. [attribute]
    Selects elements that have the specified attribute, regardless of its value.
      Example: $("input[required]") selects all  elements that have the "required" attribute.
2. [attribute=value]
    Selects elements that have the specified attribute with a specific value.
      Example: $("input[type='text']") selects all  elements with type="text". 
3. [attribute!=value]
    Selects elements that do not have the specified attribute with a specific value.
      Example: $("input[type!='text']") selects all  elements where the type attribute is not "text".  
4. [attribute^=value]
    Selects elements whose specified attribute value begins with a certain string.
      Example: $("a[href^='https']") selects all  elements where the href attribute value starts with "https".
5. [attribute$=value]
    Selects elements whose specified attribute value ends with a certain string.
      Example: $("img[src$='.png']") selects all  elements where the src attribute value ends with ".png".
6. [attribute*=value]
    Selects elements whose specified attribute value contains a certain substring.
      Example: $("a[href*='example']") selects all  elements where the href attribute value contains the substring "example".

Attribute Selectors in jQuery

jQuery lets you select elements based on their attributes and attribute values, just like CSS.
    The general syntax is:
        $("[attribute='value']")

Types of Attribute Selectors

1. Has Attribute Selector
$("[title]").css("color", "blue");  // Selects all elements that have a title attribute, regardless of its value.

2. Exact Match ([attr='value'])
$("input[type='text']").css("background", "lightyellow"); // Selects  elements where type="text".

3. Not Equal ([attr!='value'])
$("input[name!='username']").css("border", "1px solid red");  // Selects all  except those with name="username".

4. Starts With ([attr^='value'])  
$("a[href^='https']").css("color", "green");  //  Selects links where href starts with "https".

5. Ends With ([attr$='value'])
$("img[src$='.png']").css("border", "2px solid blue");  // Selects images where src ends with .png.

6. Contains ([attr*='value'])
$("a[href*='google']").css("font-weight", "bold");  // Selects links where href contains "google" anywhere.

7. Multiple Attribute Selector
$("input[type='text'][name='email']").css("background", "lightgreen");  // Selects  elements that have both type="text" and name="email".

Summary:
-----------
[attr] → has attribute
[attr='value'] → exact match
[attr!='value'] → not equal
[attr^='value'] → starts with
[attr$='value'] → ends with
[attr*='value'] → contains
[attr1][attr2] → multiple attributes
