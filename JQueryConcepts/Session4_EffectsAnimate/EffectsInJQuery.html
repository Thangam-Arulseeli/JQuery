JQuery Effects 

      jQuery provides several built-in methods to create visual effects on HTML elements. 
      These effects can enhance user experience by adding animations and transitions to your web pages. 
    
      Here are some commonly used jQuery effects:
ðŸ”¹   1. Show / Hide / Toggle
        show() â†’ Makes a hidden element visible.
        hide() â†’ Hides a visible element.
        toggle() â†’ Switches between show and hide automatically.

ðŸ”¹ 2. Fade Effects
        fadeIn() â†’ Fades in the element (increasing opacity).
        fadeOut() â†’ Fades out the element (decreasing opacity).
        fadeToggle() â†’ Switches between fade in and fade out.

ðŸ”¹ 3. Slide Effects
        slideUp() â†’ Slides the element up (hides).
        slideDown() â†’ Slides the element down (shows).
        slideToggle() â†’ Switches between slide up and down.

ðŸ”¹ 4. Custom Animations
        animate() â†’ Allows you to create custom animations by changing CSS properties over time.
        Example: $("#myDiv").animate({ left: '250px', opacity: '0.5' }, 1000);

ðŸ”¹ 5. Chaining Effects
          You can chain multiple effects together for more complex animations.
          Example: $("#myDiv").fadeOut(1000).slideUp(1000).fadeIn(1000);

ðŸ”¹ 6. Speed Options 
          You can control the speed of the effects using predefined strings ("slow", "fast") or milliseconds (e.g., 1000 for 1 second).
          Example: $("#myDiv").fadeIn("slow");
          Example: $("#myDiv").fadeIn(2000); // 2 seconds

ðŸ”¹ 7. Callback Functions  
        Many effect methods accept a callback function that runs after the effect completes.
        Example: $("#myDiv").fadeOut(1000, function() { alert("Fade out complete!"); });

ðŸ”¹ 8. Stopping Effects  
        stop() â†’ Stops the currently running animation on the selected elements.
        Example: $("#myDiv").stop();

ðŸ”¹ 9. Queueing Effects  
          By default, jQuery queues effects to run one after another. 
          You can control this behavior using the queue parameter.
          Example: $("#myDiv").fadeIn().fadeOut(); // Runs one after the other
          Example: $("#myDiv").fadeIn().fadeOut().stop(true, true); // Stops and clears the queue

ðŸ”¹ 10. Easing Effects 
          Easing functions control the acceleration of the animation. jQuery UI provides additional easing options.
          Example: $("#myDiv").animate({ left: '250px' }, { duration: 1000, easing: 'easeOutBounce' });

show()
      In jQuery, the second argument of .show() specifies speed. Instead of "slow", you can use:
    Built-in speed options
        "fast" â†’ 200ms
        "slow" â†’ 600ms
        "normal" (or no argument) â†’ default
  
        Milliseconds (custom duration)
        ---------------------------------
        You can specify your own duration in milliseconds:

        $("#box").show(1000); // 1 second
        $("#box").show(2000); // 2 seconds

        With easing (if jQuery UI is included)
        -----------------------------------------
        $("#box").show(1000, "linear");
        $("#box").show(1000, "swing");

        With callback after animation finishes
        -----------------------------------------
        $("#box").show(1000, function() {
            alert("Box is now visible!");
        });
        Example replacing "slow" with other options:
        ---------------------------------------------
        $("#btnShow").click(function() {
            $("#box").show("fast");   // or .show(1500); or .show("normal");
        });

.fadeIn() in jQuery works almost the same way as .show(), but instead of just revealing the element, 
    it gradually increases its opacity from hidden to fully visible.

      Options for .fadeIn()
      1. Predefined speeds
      $("#box").fadeIn("fast");   // 200ms
      $("#box").fadeIn("slow");   // 600ms

      2. Custom duration (milliseconds)
      $("#box").fadeIn(1000);  // 1 second
      $("#box").fadeIn(2000);  // 2 seconds

      3. With easing (if jQuery UI is available)
      $("#box").fadeIn(1000, "linear"); 
      $("#box").fadeIn(1000, "swing"); 

      4. With a callback function
      $("#box").fadeIn(1000, function() {
          alert("Fade-in complete!");
      });


Example combining everything:

$("#btnShow").click(function() {
    $("#box").fadeIn(1500, "swing", function() {
        console.log("Box faded in!");
    });
});

.fadeTo() in jQuery.

Unlike .fadeIn() / .fadeOut(), which only show or hide,
.fadeTo() lets you animate the element to a specific opacity between 0 (invisible) and 1 (fully visible).

Syntax
$(selector).fadeTo(speed, opacity, easing, callback);

ðŸ”¹ Options
Speed (duration)
"fast" â†’ 200ms
"slow" â†’ 600ms
"normal" (default) â†’ 400ms

Or a custom duration in milliseconds:
$("#box").fadeTo(1000, 0.5); // fade to 50% opacity in 1 sec


Opacity (required)
Value between 0.0 and 1.0

Example:
$("#box").fadeTo("slow", 0.3); // fade to 30% visible


Easing (optional, requires jQuery UI)
"swing" (default) or "linear"

Example:
$("#box").fadeTo(2000, 0.8, "linear");


Callback (optional)
Function to run after animation completes
$("#box").fadeTo(1500, 0.2, function() {
    alert("Fading finished, opacity is now 20%!");
});

âœ… Example combining all options
$("#btnFade").click(function() {
    $("#box").fadeTo(2000, 0.4, "swing", function() {
        console.log("Box faded to 40% opacity.");
    });
});
