Select Multiple Items:

   selectable() is a jQuery UI interaction method (not part of core jQuery).
   It lets you make a group of elements selectable with the mouse, 
         similar to selecting multiple files/folders in Windows Explorer.

    You can click and drag a box around elements, and they will be marked as selected.

    How it works:
      Hold mouse click + drag over list items → multiple items get selected.
      Selected items get the class .ui-selected (blue background in this example).
      The stop event fires after selection is complete → we list all selected items.
