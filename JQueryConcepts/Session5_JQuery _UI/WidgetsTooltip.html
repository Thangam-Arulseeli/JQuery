UI WIDGET-TOOLTIP: 
    The native tooltip is replaced by jQuery UI tooltip widget
    because the jQuery UI tooltip allows customization and adds new themes.
    Tooltips are used with the element to display a title in the title box next to the element, 
    when you hover the element with your mouse.
SYNTAX: 
$(selector, context).tooltip (options)
$(selector, context).tooltip ("action", [params])
OPTIONS: 
content,disabled,hide,items,position,show,tooltipClass,track
ACTION: 
close,destroy,enable,disable,open,option,widget
EVENT METHOD: 
create(event,ui),close(event,ui),open(event,ui)
