What is sortable()?

         sortable() is a jQuery UI interaction method (not part of core jQuery).
         It lets you reorder a list of items using drag and drop.
         Very useful for making to-do lists, dashboards, menus, or tables where users can rearrange items.

         Syntax
         $(selector).sortable(options);

         selector → the container with child items you want to reorder.
         options → (optional) settings to customize behavior.

         Common Options
         axis: "y" → restrict sorting to vertical direction.
         axis: "x" → restrict sorting to horizontal direction.
         handle: ".drag-handle" → only allow dragging by a specific handle.
         placeholder: "highlight" → placeholder style when dragging.
         connectWith: "#otherList" → drag between multiple lists.

         Example:

         $("#sortable").sortable({
         axis: "y",
         placeholder: "ui-state-highlight"
         });

         Events

         sortstart → when dragging starts.
         sortstop → when dragging stops.
         update → when the order is changed (useful to save the new order in DB).

         Example:

         $("#sortable").sortable({
         update: function(event, ui) {
            var order = $(this).sortable("toArray");
            console.log("New order: " + order);
         }
         });


         Difference Between selectable() vs sortable()
         --------------------------------------------------
         Feature	          selectable()	                              sortable()
         -----------------------------------------------------------------------------------------
         Purpose	          Select multiple items with                  Rearrange items via drag and drop
                                 mouse drag (like selecting files)	
         Action	          Highlight items	                           Change order of items
         Common use case	File selection, tagging	                     To-do lists, menu ordering
