What is selectable()?

            ** selectable() is a jQuery UI interaction method (not part of core jQuery).
            ** It lets you make a group of elements selectable with the mouse, 
                  similar to selecting multiple files/folders in Windows Explorer.
            ** You can click and drag a box around elements, and they will be marked as selected.

            Syntax
            $(selector).selectable(options);

            selector → the container or list of items you want to make selectable.
            options → (optional) settings to control the behavior.

            Common Options
            ---------------
            filter → restrict selection to certain child elements.
            cancel → prevent selection of some elements.
            tolerance → defines how an item is selected (e.g., "touch" vs. "fit").
            delay Option
               Meaning: Minimum time (in milliseconds) the mouse must be held down before the selection starts.
               Use case: Prevents accidental selection when you just click quickly.
            distance Option
               Meaning: Minimum distance (in pixels) the mouse must move while holding down before the selection starts.
               Use case: Prevents selection from starting on tiny/accidental movements.

            Example:
                  $("#selectable").selectable({
                  filter: "li",         // only list items selectable
                  tolerance: "touch"    // selected if touched by drag box
                  });

            Example:
            $("#selectable").selectable({
                  delay: 200   // must hold mouse down for 200ms before selection starts
                  });

            Example:
            $("#selectable").selectable({
                  distance: 20   // must drag at least 20px before selection begins
                  });
                  
            Events
            --------
            You can bind to events like:
            selectableselected → when an item is selected.
            selectablestart → when selection starts.
            selectablestop → when selection stops.

                  $("#selectable").selectable({
                  stop: function() {
                     var result = [];
                     $(".ui-selected", this).each(function() {
                        result.push($(this).text());
                     });
                     alert("You selected: " + result.join(", "));
                  }
                  });

            In short:
            selectable() is a jQuery UI method that makes elements multi-selectable using the mouse.
