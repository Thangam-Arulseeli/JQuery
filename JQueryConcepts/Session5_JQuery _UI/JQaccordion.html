JQUERY UI

    JqueryUI is the most popular front end frameworks currently.  
    It is sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development.   
    It uses HTML, CSS and Javascript.
    JqueryUI is a powerful Javascript library built on top of jQuery JavaScript library.     
    UI stands for User interface.   
    It is a set of plug-ins for jQuery that adds new functionalities to the jQuery core library.
    The set of plug-ins in JqueryUI includes interface interactions, effects, animations, widgets, and themes 
        built on top of jQuery JavaScript Library.
    It was released in September 2007, announced in a blog post by John Resig on jquery.com. 
    The latest release, 1.10.4, requires jQuery 1.6 or later version.   
    jQuery UI is a free, open source software, licensed under the MIT License.

    Features:
       * JqueryUI is categorized into four groups, interactions, widgets, effects, utilities. 
       *  Interactions: Draggable, Droppable, Resizable, Selectable, Sortable.
       *  Effects: Add effects and animations to your web pages.
       *  Utilities: Position, Widget factory, Theming.
       * Widgets: Pre-built, themeable UI components for your web applications.

        JqueryUI is built on top of the jQuery library, so you need to include jQuery before including JqueryUI.
        You can include JqueryUI in your web pages using a CDN (Content Delivery Network) or by downloading it and hosting it yourself.
        You can include the jQuery UI CSS file to style your UI components.
        You can also include the jQuery UI JavaScript file to add interactivity to your UI components.
        You can include the jQuery UI theme CSS file to change the look and feel of your UI components.

       * JqueryUI provides a wide range of UI components, including accordions, datepickers, dialogs, sliders, tabs, and more.
        
        Each UI component is highly customizable and can be configured using various options and methods.
        JqueryUI also provides a theming framework that allows you to create your own custom themes for your UI components.
        You can use the ThemeRoller tool to create custom themes for your UI components.  
        You can download the entire library or just the components you need.
        You can also customize the theme of your UI components using the ThemeRoller tool.
        
        JqueryUI is compatible with all major browsers, including Chrome, Firefox, Safari, Opera, and Internet Explorer 9+.
        It is also compatible with mobile browsers, including iOS and Android.
        JqueryUI is easy to learn and use, especially if you are already familiar with jQuery.
       
        Widgets are the most important part of JqueryUI.
        Widgets are the elements of the graphical user interface (GUI) that displays information or provides a
             specific way for a user to interact with the application.

        Widgets are built using a combination of HTML, CSS, and JavaScript.
        Widgets in JqueryUI are:
       *  Accordion, Autocomplete, Button, Datepicker, Dialog, Menu, Progressbar, Selectmenu, Slider, Spinner, Tabs, Tooltip.

ACCORDION

        Accordion Widget in jQueryUI is a jQuery based expandable and collapsible content holder 
            that is broken into sections and probably looks like tabs. 

        jQueryUI provides accordion() method to achieve this.

        Syntax
            $(selector, context).accordion (options);
